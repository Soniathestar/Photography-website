# Photography-website
Every moments tells a story and through my lens, those stories come alive!

## Project Description
Sonia's Gallery is a personal photography website dsigned to showcase memorable moments captured through the lens. The website hightlights the beauty of everyday scenes,tells stories behind selected photographs and introduces the photographer Sonia. It also includes a sign-up form to allow visitors to connect and engage with the site
This project focuses on visual storytelling and a simple user experience while practicing foudational web development skills.
## Tecnologies used
html:5- used to structure the page and the main content
CSS- used to style the page and the layout in general
Unsplash - used to obtain the good photographs on the website

## Project Structure
Photography-website_index.html_style.css_profile.jpg.jpeg_README.md_LICENSE

## Website sections
1. Header
This displays the website title 'SONIA'S GALLERY'
2. Introductory section
A brief description explaining the meaning of photography to the photographer Sonia and how it preserves memories through visual storytelling
3. Featured photograph
Highlights a meaningful image with a background story explaining the inspiration and emotion behind it.
4. About Sonia
Introduces the photographer with fun facts, background information, and about the photographer Sonia's journey in photography
5. Sign-up Form
Allows website visitors to enter their names as well as their email and message while accepting terms and conditions
6. Footer message
A short thank you note to the web visitors

## Installation
1. Clone or download the repository
2. Open the project folder
3. Open the index.html in any web browser
4. Ensure the style.css is also in the same directory
